GitHub手机应用：好看，但不太好用

GitHub，是一个开源项目托管仓库，也是各路开源大佬聚集的地方，我有幸参加了GitHub Mobile的内部测试，并且经历了GitHub Mobile从测试到v1.0.0的飞跃

客户端UI

客户端UI整体不错，有暗色和明色的选择，自带代码高亮功能，对于Markdown的渲染也很不错

基本功能

因为是第一个版本，所以功能肯定没有已经成熟的OpenHub多，现在它可以使用的功能有：搜索，浏览repo、查看代码、Star、Watch、Issue的大部分功能、合并请求等，虽然功能和OpenHub比少很多，但是他可以在IOS使用（OpenHub没有IOS版本），这是大部分Github App不能实现的
